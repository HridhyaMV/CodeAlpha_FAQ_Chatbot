# AIML FAQ Chatbot

## Introduction

AIML FAQ Chatbot is a web-based chatbot developed using Python and Flask. The chatbot is designed to answer frequently asked questions related to Artificial Intelligence and Machine Learning. It uses Natural Language Processing (NLP) techniques to understand user queries and provide the most relevant response from a predefined FAQ dataset.

## Project Objective

The main objective of this project is to build an intelligent FAQ chatbot that can assist students and beginners in learning AIML concepts. The chatbot reduces the need for manual searching by providing quick and accurate answers to common questions.

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* NLTK
* TF-IDF Vectorizer
* Cosine Similarity
* Session Management

## Features

* AIML related FAQ question answering
* Natural Language Processing using NLTK
* TF-IDF based text vectorization
* Cosine Similarity based question matching
* Voice Input using Speech Recognition
* Text-to-Speech response feature
* Copy Response functionality
* New Chat option
* Interactive and user-friendly interface
* Chat history management using Flask Sessions

## Working

1. The user enters a question through text or voice input.
2. The question is preprocessed using NLTK tokenization.
3. FAQ questions are converted into TF-IDF vectors.
4. Cosine Similarity is used to compare the user's question with stored FAQ questions.
5. The most relevant answer is selected and displayed.
6. Users can copy the response or listen to it using the text-to-speech feature.

## Sample Questions

* What is AIML?
* What is Machine Learning?
* What is NLP?
* What is TensorFlow?
* What projects can AIML students do?

## Conclusion

This project demonstrates the practical implementation of Natural Language Processing techniques such as tokenization, TF-IDF Vectorization and Cosine Similarity for building an intelligent FAQ chatbot. It provides a simple and effective way for users to get AIML-related information through an interactive interface.
